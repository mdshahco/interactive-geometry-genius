Q-1: What is DOM? What is the purpose of DOM?

Answer: DOM standard Document Object Model. Document object represents the whole html document. DOM main purpose of use get, change, add or delete html elements.


Q-2: How to select Element to DOM?

Answer: 5 ways to select elements in DOM.

  1. getElementsByClassName()
  2. getElementById()
  3. getElementsByTagName
  4. querySelector()
  5. querySelectorAll()

Q-3: What is the Event bubble?

Answer: Event bubble is a concept in the DOM. It is also a type of event propagation. It's a way that events are handled in the browser.


Q-4: What is the callback function? What is the purpose of callback function?

Answer: A callback is a function passed as an argument of another function. Callback function main purpose is to execute code in response to an event.
